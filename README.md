# Веб-приложение «Расписание занятий»

Данный проект представляет собой веб-приложение для просмотра расписания занятий студентов.
Проект разработан в рамках курсовой работы.

Пользователь может выбрать свою учебную группу и день недели,
после чего система отображает соответствующее расписание занятий,
получаемое из базы данных.

---

## Используемые технологии
- Java
- Spring Boot
- PostgreSQL (облачная база данных Neon)
- HTML / CSS
- Thymeleaf
- SQL

---

## Функциональные возможности
- Выбор учебной группы
- Выбор дня недели
- Просмотр расписания занятий
- Получение данных из реальной SQL базы данных
- Минималистичный и понятный пользовательский интерфейс

---

## Структура проекта
- `src/main/java` — серверная часть приложения (Spring Boot)
- `src/main/resources/templates` — HTML-шаблоны страниц
- `src/main/resources/static` — стили CSS
- `application.properties` — настройки подключения к базе данных
- `pom.xml` — конфигурация Maven
